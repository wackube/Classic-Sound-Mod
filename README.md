# Classic Sound Mod
My attempt to change a large majority of TF2 sounds into sounds from classic games such as:
- Half-life 2
- Half-life
- Counter Strike: Source
- Team Fortress: Classic
- Day of Defeat: Source
- Portal 2
- Portal
